# Experiments

Future benchmark runs for `frontier-engine` should live here.

Recommended shape:

- `experiments/<benchmark-id>/README.md`
- `experiments/<benchmark-id>/config/`
- `experiments/<benchmark-id>/runs/`
- `experiments/<benchmark-id>/analysis/`

Each benchmark should define one exact target and one measurable finite gap.

Examples now in scope:

- `m8-sat-rail`
- `p848-anchor-ladder`
