# Problem Research Bundles

Each problem can eventually grow a deeper public research bundle here:

- `research/problems/<id>/README.md`
- `research/problems/<id>/boards/`
- `research/problems/<id>/notes/`
- `research/problems/<id>/wave-history/`

These bundles complement `problems/<id>/` and `packs/<family>/problems/<id>/`; they do not replace them.
