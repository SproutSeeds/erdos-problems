The first deep harness pack in `erdos-problems`.

- `20`
- `536`
- `856`
- `857`

- `20`: deep harness, strong / uniform core
- `857`: deep harness, weak / non-uniform core
- `536`: dossier bridge, natural-density LCM analogue
